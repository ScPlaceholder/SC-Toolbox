SECTION 2 - SLOT EXTRACTOR EDGE CASE AUDIT
=======================================================================

All 76 item types across 208 ship loadouts were audited. Edge cases
verified against the Star Citizen Wiki API (EMP, TractorBeam, QIG,
BombLauncher, UtilityTurret types).

--- 2A. CONFIRMED CORRECT BEHAVIOURS ---

  BombLauncher / MissileLauncher dual-typing:
    Missile rack ports list BOTH types in itemTypes. The extractor
    correctly surfaces MissileLauncher slots and does not suppress them.
    VERIFIED CORRECT.

  QuantumInterdictionGenerator - Hornet Mk II belly port:
    QIG port has BallTurret subtype -> is_housing=True -> extractor
    recurses into children -> finds S5 WeaponGun slot. The Heartseeker
    inner gun slot is preserved. VERIFIED CORRECT.

  EMP Devices (tmbl_emp_*):
    Not in any accept_types. Ports skipped by _SKIP_PREFIXES: tmbl_emp.
    EMP slots do not appear as weapon slots. VERIFIED CORRECT.

  Mining/Tractor/Salvage Turrets:
    Skipped via _SKIP_PORT_PATTERNS: tractor, mining, salvage.
    Ship-mounted tractor beams (grin_tractorbeam_s1/s2) covered by
    erkul_supplement.json and skipped via grin_tractorbeam prefix.
    VERIFIED CORRECT.

  Mule 'Powerplant - Power' batteries:
    Non-standard type string does not match accept_types 'PowerPlant'.
    Correctly not extracted as a power plant slot. VERIFIED CORRECT.

  SureGrip TH2 Tractor Beam:
    Wiki misclassification has no effect -- Erkul has it as TractorBeam
    and it never appears in a WeaponGun loadout. VERIFIED CORRECT.

--- 2B. SKIP PATTERNS (reference) ---

  _SKIP_PORT_PATTERNS:  camera, tractor, self_destruct, landing,
    fuel_port, fuel_intake, docking, air_traffic, relay, salvage,
    mining, scan, torpedo_storage

  _SKIP_PREFIXES:  controller_, bmbrck_, mount_gimbal_, mount_fixed_,
    turret_, relay_, vehicle_screen, radar_display, grin_tractorbeam,
    tmbl_emp, umnt_, gmisl_

--- 2C. TURRET HOUSING LOGIC ---

  BallTurret is in _TURRET_HOUSING_SUBTYPES, so ports with this subtype
  recurse into children. This correctly handles the Hornet belly QIG
  port (BallTurret wrapping an S5 gun slot). Wiki confirms QIG devices
  are S3-S5 interdiction items, not weapon slots.
